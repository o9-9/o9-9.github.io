## Features

- 🚀 Built with [Astro](https://astro.build/) v5.12.9 for optimal performance
- 💨 Styled with [TailwindCSS](https://tailwindcss.com/) v4.1.11
- 🌙 Dark/Light mode toggle with system preference detection
- 📱 Fully responsive design with mobile menu
- ♿ Accessible with keyboard navigation and ARIA attributes
- 🔤 Custom JetBrains Mono font
- 📄 Clean, semantic HTML
- 📦 Optimized SVGs and assets
- 📰 RSS feeds

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/o9-9/o9-9.github.io

# Navigate to the project directory
cd o9-9.github.io

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

### Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
